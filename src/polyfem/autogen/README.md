# Generating bases

Dependencies

- `sympy`
- `numpy`

## Tri/Tet Bases

```python
python p_bases.py ./
```

This generates `auto_p_*` files

## Tri/Tet Bernstein Bases

```python
python p_bases.py ./ --bernstein
```

This generates `auto_b_*` files

## Quad/Hex Bases

```python
python q_bases.py ./
```

This generates `auto_q_*` files
